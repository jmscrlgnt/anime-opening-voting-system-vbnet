# Anime Opening Voting System (VB.NET)

## 📌 Overview
A desktop-based voting system built with VB.NET that allows users to vote for their favorite anime openings. The system includes user authentication, admin controls, and vote management.

## 🚀 Features
* User Registration and Login
* Admin Login and Dashboard
* Anime Opening Voting System
* Vote Counting and Tracking
* Password Recovery System

## 🛠 Tech Stack
* Language: VB.NET
* Framework: .NET Framework (Windows Forms)
* Database: SQL Server (Local DB)
